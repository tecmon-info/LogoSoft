一般情報

CD-ROM の <ドライブ>:\Readme にある README.TXT ファイルを確認してください。

特にファイルの後半に記載されている "既知の問題" をお読みください。 この "既知の問題" には、マイクロソフトインテリマウス（Microsoft Intellimouse）の使用に関する記述があります。 マウスポインタの軌跡がオフになっていることを確認してください(スタート メニュー\設定\コントロール パネル\マウス)。

Cirrus グラフィック カード ドライバの最新版は、CD-ROM の <ドライブ>:\Windows\Tools\7548pci にも含まれています。

このフォルダには、最新版のドライバを含む 2 つの自己解凍ファイルが含まれています。 これらのファイルをハード ディスク ドライブ上の任意のフォルダ、あるいは、ダブルクリックで A ドライブのフロッピー ディスクに解凍します。この作業は必ず両ファイルに対して行ってください。 この作業で、最新版のグラフィック ドライバの入ったフロッピー ディスクを 2 枚作成します。

 

グラフィック ドライバのアップデート

デスクトップを右クリックして、メニューを表示します。

[プロパティ] → [設定] → [詳細設定] を選択します。

グラフィック ドライバのプロパティを表示するダイアログ ボックスが表示されます。

[変更] ボタンを左クリックして、現在のディスプレイ アダプタを表示します。 フロッピー ディスク ドライブに 1 枚目のディスクを挿入し、[ディスク使用] ボタンをクリックします。 続いて [OK] をクリックし、プロンプトが表示されたら、2 枚目のフロッピー ディスクを挿入します。

ファイルのコピーが完了したら、[適用] ボタンをクリックします。（重要！ここで PC を再起動しないでください。）

 

[OK] をクリックして、PC を再起動します。

 

以下の手順に従って、グラフィック ドライバが正常にアップデートされたか確認してください。

デスクトップを右クリックして、メニューを表示します。

[プロパティ] → [設定] → [詳細設定] を選択します。

グラフィック ドライバのプロパティを表示するダイアログ ボックスが表示されます。

[変更] ボタンの左側に次のような表示があれば、アップデートは成功です。
