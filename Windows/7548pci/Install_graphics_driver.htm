General Info

Check out the README.TXT file on the CD - <Drive>:\Readme.

Particularly the section marked ‘Known Problems’ at the end of the file. This refers to using  the Microsoft Intellimouse. Make sure that mouse trails are switched OFF(Start Menu\Settings\Control Panel\Mouse).

The update for the Cirrus graphics card drivers also exist on the CD - <Drive>:\Windows\Tools\7548pci.

This folder contains two self-extracting files containing the updated driver. Either extract these files to a folder on the hard drive or double click the file to extract the contents to a Floppy disk in Drive A. You must do this for both files on the CD. At the end of this process you should end up with two floppy disks containing the graphics driver update.

 

To Update the Graphics Drivers

Right click on the Desktop to bring up the menu.

Select Properties, Settings, Advanced Properties.

This brings up a Dialog Box displaying the graphics driver properties.

Left click the ‘Change’ button to display the current display adapter. Put the first disk in the floppy drive and click the ‘Have Disk’ button. Click ‘OK’ and insert the second disk when prompted.

Once the file copying process has finished, click on the ‘Apply’ button (This is most important - Do not rely on re-booting the PC to update the drivers)

 

Click on ’OK’ and restart PC.

 

Check that graphics drivers have been successfully updated i.e.

Right click on the Desktop to bring up the menu.

Select Properties, Settings, Advanced Properties.

This brings up a Dialog Box displaying the graphics driver properties.

To the left of the ‘Change’ button should read:-

Cirrus Logic 7548 1.40D PCI(Cirrus)
